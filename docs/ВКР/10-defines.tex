\begin{description}
    \item[Индекс] Абстрактная структура данных, создаваемая с целью повышения операций поиска по заданному массиву за счет повышения затрат на хранение. Примеры: в СУБД: b-tree, в Python: dict, в C++: map.

\end{description}
